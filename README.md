# Advanced POS System

## English
Advanced POS ready-made project for mini markets and shops with admin CRUD, sales report, stock management, cart, receipt print, localStorage, and bilingual support.

## မြန်မာ
ဒီ project သည် မီနီမားကတ်နှင့် စတိုးဆိုင်များအတွက် အသင့်သုံးနိုင်သော Advanced POS System ဖြစ်ပြီး အောက်ပါ features များပါဝင်ပါသည်။

### Features
- Login Page
- Admin Product Management
- Add / Edit / Delete Product
- Product Search
- Add to Cart
- Checkout
- Receipt Print
- Sales Report
- Stock Management
- LocalStorage Save
- Burmese / English
- MMK / Baht
